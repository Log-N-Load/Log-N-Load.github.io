Command Injection {#command-injection}

Command injection se trata de la ejecución de comandos del sistema operativo a través de entradas no válidas, afectando la seguridad del servidor, sistema y red.

  • Vector

Si se modifica el valor del input y se concatena un comando, en este caso “ipconfig” devuelve el contenido del fichero y el resultado del comando.

  • Vulnerabilidad

El input no se sanitiza, permitiendo la ejecución de código en el servidor.

  • Solución/Mitigación

Existen varias soluciones, en este caso, como ya se sabe a qué archivos puede acceder, se crea una whitelist.

Todo lo que no esté dentro de la whitelist devolverá un error.

Al intentar atacar de la misma forma recibe “FIle not found”.

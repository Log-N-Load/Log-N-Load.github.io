Carving

El proceso de excarvar entre los diferentes sectores de un disco duro para recuperar información.

# Se descarga la herramienta “Photorec” incluida en la herramienta “testdisk”
Para ejecutar la herramienta, dentro de la carpeta se ejecuta “sudo ./photorec_static”

Una vez se inicie mostrará un menú donde se podrá seleccionar el dispositivo de donde se quieren recuperar los archivos.

Se selecciona la partición o el disco entero

Se elige el tipo de sistema de archivos que usa el dispositivo (Se puede ver con “lsblk -f)

Se selecciona si se quieren buscar archivos de espacio no asignado o de todo el disco

La carpeta donde extrae los archivos recuperados

El proceso comenzará tras seleccionar la carpeta, la duración puede variar dependiendo del tamaño y estado del dispositivo.

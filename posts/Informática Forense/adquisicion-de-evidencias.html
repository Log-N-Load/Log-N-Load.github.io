Adquisición de evidencias

El primer paso es prevenir que se pueda contaminar la evidencia escribiendo o modificando los datos. Para ello se utiliza un SO creado para análisis forense como CAINE que por defecto no tiene escritura en dispositivos externos, o prevenir que los dispositivos externos se monten automáticamente por defecto al conectarlos al SO.
Se ha optado por la segunda opción.

Para prevenir que Ubuntu monte dispositivos externos automáticamente, se escriben lo siguientes comandos en la consola

gsettings set org.gnome.desktop.media-handling automount false
gsettings set org.gnome.desktop.media-handling automount-open false

Una vez se ha evitado que el SO monte los dispositivos, se conecta un USB y se crea la imagen, para ello se utiliza el comando “dd”.

Sin necesidad de montar el dispositivo, se escribe el comando “dd if=Carpeta/Origen of=Carpeta/Destino.dd status=progress”.
El comando “lsblk” muestra dónde está el dispositivo origen.

Una vez terminado el proceso se saca el hash del resultado

Una vez adquirida la evidencia se puede trasladar al laboratorio. Es recomendable crear varias copias de la imagen y trabajar sobre ellas.

Análisis MFT

Para analizar el MFT se necesita un disco o una partición formateada en NTFS. En este caso se ha creado una partición de 5GB para agilizar el proceso de análisis.

Una vez creada la partición, se inicia la herramienta “Mft2Csv”, esto sacará la tabla y la pasará a formato csv.
Para ello, se especifica la partición y si se desea, se puede cambiar el separador para el csv.

Tras el proceso de extraccion de MFT, se creá una carpeta con todo el contenido. Donde se puede acceder a la tabla y verificar la información.

Cuando se borra un archivo aparecen 2 archivos, uno con metadatos y otro con los datos. (Con nombre $I y $R y flag hidden)
Al eliminar definitivamente solo muestra un archivo.

Para extraer información de un solo registro o archivo se necesita la herramienta “MFTRCRD”, una vez se obtiene la herramienta se abre la consola como administrador y se ejecuta el comando “MFTRCRD64.exe path\&file -d indxdump=off 1024 -s > pruebaMFTRCRD.txt”


Una vez finalizado el proceso se podrá ver el resultado accediendo al fichero de salida especificado
